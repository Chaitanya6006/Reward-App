# 🌟 Reward App

## Introduction to Reward App :
Reward App is a decentralized task-based dApp built on Stellar Soroban that allows users to complete tasks and earn rewards directly into their wallet using Freighter. ✨  

It ensures transparency, on-chain reward distribution, and an interactive user experience.

---

## Pre-Requisites to Run :
- Download Freighter Wallet web extension  
- Setup your Freighter Wallet  
- VS Code should be installed  
- Node.js & npm installed  

---

## How To Run :

open in VS code ----> Go to Terminal ----> cd frontend ----> Give a command "npm run dev" ----> Run on Localhost (CTRL + click) -----> You're Ready To Go 🚀  

or  

simply click on deployed dapp link (deployed via vercel)

## Inter Contract Call :

The inter-contract call in this project is used during reward distribution.  

- The smart contract interacts with a token contract  
- It transfers reward tokens to the user after completing tasks  

📌 Logic:
- Fetch token contract address  
- Initialize Soroban token client  
- Execute transfer function  

This ensures secure and on-chain reward processing.
